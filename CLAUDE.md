# Smriti — AI Instructions

This repository is **Smriti (स्मृति)** — Devalok's institutional knowledge base.

## What this repo is

The single source of truth for everything about Devalok Design and Strategy Studio — its identity, philosophy, processes, culture, terminology, brand guidelines, services, and operational details. This is NOT a codebase.

## How to use it

- **Start with `core/devalok-complete-context.md`** — this is the master reference document. It contains the complete, integrated view of Devalok.
- **For detailed deep-dives**, check the standalone files in `glossary/`, `services/`, `publications/`, etc. These contain more detail than the complete context doc carries.
- **When updating content**, always check if the content also exists in the complete context doc. If it does, update both in the same commit. See the README for the source-of-truth rules.

## Versioning

Each document has a version in its frontmatter (e.g., `version: 1.0.0`). Bump appropriately:
- Patch: corrections, clarifications
- Minor: new sections, significant content updates
- Major: structural rewrites

## Key conventions

- This repo is about **Devalok the studio**, not any specific product
- No code belongs here
- Use the Devalok voice: warm, story-driven, rooted in tradition, confident but humble
- Sanskrit/Hindi terms are used intentionally — never for decoration
- When in doubt about tone or terminology, the complete context doc is the guide
